# Gelato Limit Orders

- Mainnet: https://thegraph.com/explorer/subgraph/gelatodigital/limit-orders
- Ropsten: https://thegraph.com/explorer/subgraph/gelatodigital/limit-orders-ropsten

### Install

```bash
yarn install
```

### Deploy mainnet

```bash
yarn prepare:mainnet

yarn deploy:mainnet
```

### Deploy ropsten

```bash
yarn prepare:ropsten

yarn deploy:ropsten
```
